# TechSpace
###An event app for the tech community
--
#User Stories

1. A browser can explore upcoming tech-related events.
2. A browser can post a new event.
3. A browser can search through events by keyword and date.

#Technologies
- Node.js
- Express
- Mongoose
- MONGODB
- JavaScript
- JQuery
- CSS
- HTML
